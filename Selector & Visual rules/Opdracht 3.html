Er zijn verschillende selectors, bijvoorbeeld;
  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • Een Typ selector selecteerd de elementen in een html document met dezelfde.
    Voorbeeld: span {background-color: blue}. Alle span elementen zullen nu een blauwe achtergrond hebben.
    Voordeel is dat veel dezelfde elementen in 1 keer gestyled kunnen worden.
  • Een descendant selector komt overeen met alle elementen die afstammelingen zijn van een opgegeven element.
    Voorbeeld: div p {Background-color: yellow}. Nu worden alle p elementen binnen div elementen geselecteerd.
    Voordeel: zo kunnen er stijlen toegepast worden op specifieke gebieden van de pagina.
  • Bij een class selector worden elementen geselecteerd met een specifiek class-attribuut.
    Voorbeeld: .red {color: red;}. Nu is er een class gemaakt met bijbehorende declaratie. Wanneer in het document nu de class red wordt gebruikt in een element, zal het element de eigenschappen krijgen die gedeclareerd zijn in de class.
    Voordeel: Class selectors kunnen in meerdere, verschillende elementen gebruikt worden.
