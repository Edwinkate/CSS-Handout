Opdracht 1
Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  • Een externe stylesheet maken en deze koppelen aan het HTML document.
  • Een stylesheet in het HTML document maken, dit is een interne stylesheet.
  • De stijl kan verandert worden in een enkele HTML-tag. Het CSS gedeelte komt in de HTML-tag.
