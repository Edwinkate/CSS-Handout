CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Een syntax bestaat uit een Selector en een Declaration waarbij de Declaration weer bestaat uit een Property en een Value.

  • De Selector verwijst naar het HTML element wat gestyled moet worden.
  • De declaration verwijst naar wat en hoe iets gestyled moet worden.
  • De Property is wat gestyled moet worden in de declaration.
  • De Value is de waarde in hoe iets gestyled moet worden.
Voorbeeld: h1 {color:blue}.
  • h1 is hier de selector.
  • {color:blue} is de declaration.
  • color is de property.
  • blue is de value.
