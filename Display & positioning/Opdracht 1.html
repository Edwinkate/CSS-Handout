Opdracht 1
Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

  • Static: Standaardwaarde. Elementen worden op volgorde weergegeven, zoals ze in het document verschijnen.
  • Absolute: Het element is gepositioneerdten opzichte van zijn eerste gepositioneerde bovenliggend element
  • Fixed: Het element is gepositioneerd ten opzichte van het browservenster
  • Relative: Het element is relatief gepositioneerd ten opzichte van de normal positie, dus "left:20px" voegt 20 pixels toe aan de LINKER-postitie van het element.
  • Sticky: Het element wordt gepositioneerd op basis van de scrollpositie van de gebruiker. Een sticky element wisselt tussen relatief en vast, afhankelijk van de scrollpositie. het wordt relatief gepositioneerd totdat er een bepaalde offset-positie wordt bereikt in de viewport- dan blijft het op zijn plaats "plakken".
  • Initial: Stelt deze eigenschap in op de standaardwaarde.
  • Inherit: Erft deze eigenschap van het bovenliggende element.
